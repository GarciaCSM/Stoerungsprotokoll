# Störungsprotokoll (React Native / Expo)

Dieses Repository enthält ein kleines Starter-Template für eine React Native App (Expo Managed Workflow).

Voraussetzungen
- Node.js (empfohlen: 18.x oder neuer)
- npm (liegt bei Node bei)

Wie du lauffähig machst (PowerShell-Beispiele)

1) In das Projektverzeichnis wechseln

Set-Location -Path "c:\Users\MelihIskender\VS Code\Störungsprotkoll"

2) Abhängigkeiten installieren

npm install

3) Die App starten (Expo)

npx expo start

Anschließend öffnet sich Metro/Broswer, du kannst ein Android-Emulator, iOS-Simulator (Mac) oder dein echtes Gerät (Expo Go) verwenden.

Wenn du möchtest, führe ich jetzt `npm install` in deinem Workspace aus und starte `npx expo start` (oder nur die Installation). Sag mir kurz, ob ich das ausführen soll.
